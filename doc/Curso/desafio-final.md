## 🏗️ O Projeto: Dashboard de Vendas
Seu objetivo é criar um sistema que gerencia as vendas por categoria.

### 📋 Requisitos do Projeto:
1.  **Componentização**: Crie um componente `<sale-card>` para exibir cada venda separadamente.
2.  **Lista Reativa**: Use `b-for` para renderizar uma lista de vendas que pode ser editada.
3.  **Inteligência (Computed)**:
    - Calcule o **Valor Total** de todas as vendas automaticamente.
    - Calcule a **Média de Vendas** em tempo real.
4.  **Formulário Dinâmico**: Use `b-model` para adicionar novas vendas à lista.
5.  **Controle Visual**: Use `b-if` para mostrar uma mensagem de "Meta Alcançada 🏆" se o valor total ultrapassar 1000 moedas.

---

## 👁️ Dicas de Mestre:
- Use **`TJSArray`** na sua seção `data:` para que a lista de vendas atualize o DOM sempre que você der um `push`.
- Lembre-se de usar **`TJSObject`** para cada venda da lista (ex: `{ item: 'Espada', preco: 150 }`).
- Estilize seu dashboard para que pareça algo digno de um Rei! 🏰✨
