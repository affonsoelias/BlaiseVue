## 🏗️ The Project: Sales Dashboard
Your goal is to create a system that manages sales by category.

### 📋 Project Requirements:
1.  **Componentization**: Create a `<sale-card>` component to display each sale separately.
2.  **Reactive List**: Use `b-for` to render a list of sales that can be edited.
3.  **Intelligence (Computed)**:
    - Automatically calculate the **Total Value** of all sales.
    - Calculate the **Sales Average** in real-time.
4.  **Dynamic Form**: Use `b-model` to add new sales to the list.
5.  **Visual Control**: Use `b-if` to show a "Goal Achieved 🏆" message if the total value exceeds 1000 coins.

---

## 👁️ Master Tips:
- Use **`TJSArray`** in your `data:` section so that the sales list updates the DOM whenever you `push`.
- Remember to use **`TJSObject`** for each sale in the list (e.g., `{ item: 'Sword', price: 150 }`).
- Style your dashboard so it looks worthy of a King! 🏰✨
