# 📝 Módulo 01: O Início do Seu Caminho na Web Pascal
**Bem-vindo ao Treinamento Oficial BlaiseVue.**

Se você já conhece Pascal e quer construir aplicações modernas para a internet, você veio ao lugar certo! O BlaiseVue foi criado para que você não precise de anos de estudo em JavaScript para ser um mestre no frontend.

---

## 🏗️ Configuração do Ambiente
Para começar, certifique-se de que o compilador **FPC** esteja instalado e as pastas do **Pas2JS** estejam acessíveis. Se o comando `bv.exe` funcionar no seu terminal, você já está pronto para a batalha! 🛡️

## 🚀 Desafio 01: Meu Primeiro Grito Reativo
Neste módulo, vamos modificar o projeto `demo-app` para ver como o Pascal fala com o navegador.

1.  Abra a pasta `demo-app`.
2.  No terminal, digite: `..\bin\bv.exe run dev`.
3.  Abra o arquivo `src/views/Home.bv`.
4.  Mude o `titulo` de "Laboratório" para **"Meu Primeiro App Blaise!"**.
5.  Salve e atualize o navegador! ✨✨✨
