# 📝 Module 01: The Beginning of Your Web Pascal Journey
**Welcome to the Official BlaiseVue Training.**

If you already know Pascal and want to build modern web applications, you've come to the right place! BlaiseVue was created so you don't need years of JavaScript study to be a frontend master.

---

## 🏗️ Environment Setup
To get started, make sure the **FPC** compiler is installed and the **Pas2JS** folders are accessible. If the `bv.exe` command works in your terminal, you're ready for battle! 🛡️

## 🚀 Challenge 01: My First Reactive Shout
In this module, we will modify the `demo-app` project to see how Pascal speaks to the browser.

1.  Open the `demo-app` folder.
2.  In the terminal, type: `..\bin\bv.exe run dev`.
3.  Open the file `src/views/Home.bv`.
4.  Change the `titulo` from "Laboratory" to **"My First Blaise App!"**.
5.  Save and refresh the browser! ✨✨✨
