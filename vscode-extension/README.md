# 🛡️ BlaiseVue VS Code Extension

This extension provides full language support for **BlaiseVue Single File Components (.bv)**.

## ✨ Features
- **Syntax Highlighting**: HTML templates, Pascal scripts, and CSS styles.
- **Smart Brackets**: Support for Pascal `begin/end` and standard brackets.
- **Snippets**: Type `bvapp` to quickly scaffold a new component.

## 🚀 How to Install (Local)

Since this extension is in development, you can install it manually in your VS Code:

1.  **Locate your VS Code Extensions folder**:
    - **Windows**: `%USERPROFILE%\.vscode\extensions`
2.  **Create a folder** named `blaisevue-vscode` inside that directory.
3.  **Copy all files** from this `vscode-extension` folder into the new `blaisevue-vscode` folder.
4.  **Restart VS Code**.

## 🛡️ Recommended Extensions
To get the best experience with Pascal syntax highlighting, we recommend installing the following extension:
- **Pascal** (alefragnani.pascal)
