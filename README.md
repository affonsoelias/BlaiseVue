## 🚀 Como Iniciar Rapidamente

1.  Clone o repositório.
2.  Navegue até a pasta `demo-app`.
3.  Execute `..\bin\bv.exe run dev`.
4.  Abra o `dist/index.html` e sinta o poder do Pascal Reativo! ✨✨✨
